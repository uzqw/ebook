# 项目总览

青简书房是一个本地优先电子书阅读系统。它参考 `arch-reference` 的 Vue + PocketBase 架构和浅绿色侧栏风格，但领域替换为书籍阅读。

## 当前范围

1. 用户注册登录。
2. 上传 PDF、EPUB、MOBI 书籍。
3. PocketBase Go 后端解析页数、目录和每页文本。
4. 阅读页按页请求后端渲染 PNG，并同步展示“本页文本”。
5. 支持目录/页面索引跳页、书签、笔记、书籍信息和阅读记录汇总。

## 格式支持

- PDF：使用 `go-pdfium` WebAssembly 解析 outline、页数、文本、尺寸，并按需渲染 PNG。
- EPUB：使用 `go-fitz`/MuPDF 解析和渲染；解析前注入 CJK 字体，保证图片渲染和文本分页一致；目录优先读取 EPUB3 `nav.xhtml`，没有可用 nav 时回退到 `toc.ncx`。
- MOBI：使用 `go-fitz`/MuPDF 解析和渲染，目录能力按 MuPDF 输出 best effort。

## 非目标 / 暂不覆盖

- OCR、全文搜索、多人共享、批量导入。
- 对所有 DRM/损坏/非标准 EPUB 的完全兼容。
- 移动端深度适配和复杂排版编辑。

## 关键维护约定

- EPUB/MOBI 仍由后端返回页面 PNG；不要把 EPUB 简化成纯文本阅读模式。
- EPUB 文本解析和图片渲染必须使用同一份注入 CJK 字体后的临时 EPUB，否则页码会错位。
- 所有 `go-fitz` native 调用必须串行化，避免并发 SIGSEGV。
- 新的解析/渲染问题优先沉淀到 `omx_wiki/`，并同步 README 的运维说明。
